
# Kroman

Kroman is ultimate Korean hangul romanization tool. It has a lot of
implementations.

This is the python package.

## Installation

Install it with pip is recommended.

```
$ pip install kroman
```
Or install it yourself as:
```
$ python setup.py install
```

## Usage

``` python
kroman.parse(손목시계)
=> son-mog-si-gye
```
